Battle Math is a project that I dearly love. Battle math is a react app that aims to incorporate learning and video games, targeting elementary aged students. The goal is simple but I highly encourage you to create a profile and explore it yourself! It is deployed through heroku and was built in 7 days. I really wanted to foster an environment of exploration, and as you level up, you'll gain more abilities to defeat the tougher enemies. If you'd like to try it out click here!

Plot Holes was a two day project made strictly with HTML,CSS, and Javascript. It's deployed through github and was a small group project working through using external APIs. Plot Holes is a simple trivia game that uses OMDB for plots and depending on whether you win or lose, you'll receive a celebratory or condemning gif, respectively! The gifs come courtesy of the Giphy API, and was a really fun quick app to develop. If you'd like to try it out click here!
